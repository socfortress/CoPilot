## How to think about power features

Power features usually fit into one of these buckets:

- **Exposure management** (patch/vuln posture)
- **Assessment** (cloud/web scanning outputs)
- **Reporting** (shareable artifacts)
- **Acceleration** (AI assistance)

They typically have additional inputs (connectors, permissions, targets) and should be rolled out after your core ingest‚Üíalerting path is healthy.
